# Sales-Analytics-Dashboard

# 📊 Interactive Sales Analytics  Performance Dashboard

## Apex Planet Internship – Task 3

### 📌 Project Overview
This project showcases an interactive **Sales Analytics Dashboard** built using **Microsoft Power BI**. The dashboard provides valuable insights into sales performance by analyzing revenue, customer behavior, product performance, city-wise sales, age groups, and gender distribution. Interactive filters allow users to explore the data dynamically for better business decision-making.

---

## 🛠️ Tools & Technologies Used
- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Visualization
- Data Cleaning & Transformation

---

## 📂 Dataset
The dataset contains sales transaction details, including:
- Order ID
- Customer Information
- Product Details
- Category
- City
- Gender
- Age Group
- Quantity Sold
- Unit Price
- Total Revenue
- Order Date

---

## 📈 Key Performance Indicators (KPIs)
- 💰 Total Revenue
- 🛒 Total Orders
- 👥 Total Customers
- 📦 Total Quantity Sold
- 📊 Average Order Value

---

## 📊 Dashboard Visualizations
- Monthly Revenue Trend
- Top Products by Revenue
- Revenue by City
- Revenue by Category
- Revenue by Gender
- Revenue by Age Group
- Interactive Slicer (Month, Category, City, Gender, Product)

---

## 🔍 Key Insights
- Electronics generated the highest revenue among all product categories.
- Laptop is the top-performing product by revenue.
- Revenue varies across different cities, highlighting key sales locations.
- Male and female customer contributions can be compared using the dashboard.
- Customers aged **26–35** contributed the highest revenue.
- Monthly revenue trends help identify peak and low sales periods.
